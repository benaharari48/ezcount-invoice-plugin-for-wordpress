== תיאור ==
תוסף ליצירת חשבוניות באמצעות ממשק EZCount עבור הזמנות WooCommerce. פותח על ידי Dooble.

== התקנה ==
1. העלה את התיקייה `ezcount-invoicing` לתיקיית `/wp-content/plugins/`.
2. הפעל את התוסף דרך התפריט 'תוספים' בוורדפרס.
3. הגדר את מפתח ה-API, דוא"ל המפתח וכתובת ה-URL בסביבת התוסף בתפריט 'הגדרות EZCount'.

== שדות הגדרות ==
1. **API Key** - המפתח שלך לשימוש ב-EZCount API.
2. **Developer Email** - דוא"ל המפתח.
3. **Environment URL** - כתובת ה-URL של סביבת ה-EZCount (DEV: https://demo.ezcount.co.il/ PROD: https://api.ezcount.co.il/).
